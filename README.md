# Saarwood Main Site

Separate public brand site for `saarwood.ch`.

## Scope

- public brand and product landing page
- imprint, privacy, contact and release information
- links to the teleprompter app and the separate adminpanel

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment notes

- This repository is not the teleprompter app.
- The live domain should point here, not to the teleprompter monorepo.
- Keep admin functionality in the separate adminpanel repository.
